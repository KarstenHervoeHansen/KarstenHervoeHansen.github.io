The DK-Meter Online updater is an easy way to keep your unit updated with the latest improvements.Download and run the software while the DK-Meter is connected to the USB port. On Windows 10 no drivers are needed but older version of Windows will request to install the provided USB-driver. Always check for new updates and the Online updater supports DK1 to 5 and the DKT7 family.
